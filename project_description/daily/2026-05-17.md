# Resumo
> O foco desta semana foi definido para o desenvolvimento técnico, com foco total em deixar pronto a parte gráfica do projeto. A semana inicia-se no domingo e todos os membros do grupo irão documentar no mesmo arquivo o que foi realizado por cada um nas "Partes envolvidas" deste documento. 

## Desenvolvimento
O projeto foi totalmente estruturado no github, dividindo em duas branchs: "main", para etapas em que o projeto está funcional e todos os membros da equipe aprovaram as mudanças e realizaram testes e a branch "dev", para quando houver alguém editando e/ou a versão ainda não houver sido aprovada. 

## Documentação
Para a documentação, foi definido que o foco total para esta semana deve ser o tópico 2.6, que será preenchido conforme as mudanças são realizadas e oficializadas para a versão main. Estes documentos diários serão utilizados como documentação para que os membros do grupo acompanhem as mudanças.
